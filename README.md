# ☕ Nada Coffee Frontend Project

나다커피(Nada Coffee)의 신선한 원두와 음료를 만나볼 수 있는 프리미엄 커머스 플랫폼입니다.

## 📅 작업 현황 (2026-01-30)

### ✅ 오늘 완료한 작업 (Today's Achievements)

#### 1. 토스페이먼츠 결제 시스템 구축 (Core Feature) 💳
- **결제 위젯 통합**: `@tosspayments/payment-widget-sdk`를 사용하여 실제 결제 프로세스 구현.
- **모달 기반 결제창**: '결제하기' 클릭 시 화면 중앙에 보안 결제 모달이 나타나는 세련된 UX 도입.
- **렌더링 최적화**: 위젯의 `ready` 상태를 감지하여 "결제 UI 미렌더링" 에러를 완벽히 해결.
- **공용 테스트 키 적용**: 즉시 테스트 가능한 환경 구축 완료.

#### 2. 어드민 관리 시스템 안정화
- **에러 완전 정복**: 카테고리/상품 관리 시 발생하던 401(인증), 404(경로), 500(서버) 에러 해결.
- **데이터 관리 혁신**: 
  - 서버의 `limit: 100` 제한을 극복하기 위한 **Multi-fetch(다중 호출)** 로직으로 224개 전체 상품 로드 성공.
  - **전체 상품 대상 가격(Price) 정렬** 및 30개 단위 페이지네이션 구현.
  - 상품 수정 후 **이전 필터/정렬/페이지 상태를 그대로 유지**하는 스마트 리다이렉트 적용.

#### 3. 상품 목록 및 필터링 (Menu Page)
- **파일명 기반 자동 분류**: `카테고리_상품명_가격` 규칙을 인식하여 서버 버그와 상관없이 정확한 카테고리 매칭 구현.
- **디자인 고도화**: 
  - 가격 오른쪽 정렬 및 상품명 상단 정렬로 가독성 향상.
  - 구매 버튼을 GNB와 통일된 브랜드 갈색으로 변경 및 호버 효과 적용.

#### 4. 장바구니 및 주소 서비스
- **장바구니 UI**: `max-w-7xl` 확장 레이아웃, 타이틀 중앙 정렬, 버튼 1:1 비율 배치.
- **주소 검색**: `react-daum-postcode` 모달 통합으로 편리한 배송지 입력 기능 추가.

## 🛠 Tech Stack
- **Frontend**: React 19, TypeScript, Vite
- **Styling**: Tailwind CSS v4, Framer Motion
- **State Management**: Zustand, TanStack Query v5
- **Integration**: Toss Payments, Daum Postcode API
