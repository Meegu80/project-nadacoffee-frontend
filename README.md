# ☕ 나다커피 (NADA COFFEE) - 이커머스 플랫폼

나다커피는 현대적인 감각의 커피 브랜드 이커머스 플랫폼입니다. 사용자에게는 직관적인 쇼핑 경험을, 관리자에게는 강력한 데이터 분석 및 관리 도구를 제공합니다.

---

## 📋 발표용 핵심 요약 (PPT 제작 소스)

### 1. 프로젝트 개요
- **프로젝트명**: 나다커피 (NADA COFFEE) 프론트엔드
- **개발 기간**: 2025.02 ~ 2025.03
- **기술 스택**: React 19, TypeScript, Tailwind CSS, TanStack Query (v5), Zustand, Framer Motion, Toss Payments SDK

---

### 2. 핵심 기능 TOP 3 (Key Features)

#### ① 실시간 데이터 기반 '주간 베스트 셀러' 랭킹 시스템
- **특징**: 단순 판매량이 아닌 **최근 7일간의 누적 매출액**을 기준으로 순위를 산출합니다.
- **기술적 포인트**:
  - **정밀한 기준 시점**: 매일 낮 12시 정각을 기준으로 데이터를 리셋하여 최신 트렌드를 반영합니다.
  - **데이터 병합 최적화**: 서버의 호출 제한(100개)을 극복하기 위해 300개 이상의 상품 데이터를 병합 처리하여 랭킹의 정확도를 높였습니다.
  - **인터랙티브 UI**: Framer Motion을 활용한 2행 교차 마키(Marquee) 애니메이션으로 시각적 몰입감을 제공합니다.

#### ② 강력한 관리자 대시보드 및 주문 통합 관리
- **특징**: 운영 효율성을 극대화하기 위한 데이터 시각화 및 일괄 처리 기능을 제공합니다.
- **기술적 포인트**:
  - **매출 시각화**: SVG와 Framer Motion을 결합하여 주간 매출 추이를 동적인 꺽은선 그래프로 구현했습니다.
  - **일괄 액션**: 여러 주문의 상태를 한 번에 변경하거나 취소하는 벌크 액션 기능을 통해 운영 시간을 단축했습니다.
  - **실시간 재고 알림**: 품절 임박 상품을 대시보드에서 즉시 확인하고 관리할 수 있는 알림 시스템을 구축했습니다.

#### ③ 하이브리드 결제 및 포인트 시스템
- **특징**: 토스 페이먼츠와 자체 포인트 시스템을 결합한 유연한 결제 경험을 제공합니다.
- **기술적 포인트**:
  - **0원 결제 로직**: 전액 포인트 결제 시 PG사 위젯을 거치지 않고 즉시 주문을 완료하는 최적화된 프로세스를 구현했습니다.
  - **실시간 연동**: 결제 성공/취소 시 포인트 잔액과 주문 상태가 즉시 동기화되도록 React Query의 캐시 무효화 전략을 사용했습니다.

---

### 3. 기술적 도전과 해결 (Technical Challenges)
- **무한 스크롤 스크롤 튐 현상**: 데이터 추가 로드 시 스크롤 위치가 초기화되는 문제를 `useRef` 플래그와 `IntersectionObserver` 직접 구현을 통해 해결했습니다.
- **보안 강화 (XSS 방지)**: `dangerouslySetInnerHTML` 사용 시 발생할 수 있는 보안 취약점을 `DOMPurify` 라이브러리를 도입하여 해결했습니다.
- **비로그인 사용자 경험**: 관리자 API 에러 시 폴백(Fallback) 데이터를 보여주는 로직을 설계하여 모든 사용자에게 끊김 없는 화면을 제공했습니다.

## 🚀 향후 보완 및 발전 방향

1. **백엔드 분석 API 고도화**: 현재 프론트엔드에서 수행하는 복잡한 랭킹 집계 로직을 백엔드 전용 통계 API로 이전하여 클라이언트 부하를 줄일 예정입니다.
2. **모바일 앱 최적화 (PWA)**: 오프라인 모드 지원 및 푸시 알림 기능을 추가하여 모바일 사용자 경험을 네이티브 앱 수준으로 끌어올릴 계획입니다.
3. **테스트 자동화 도입**: Jest와 Cypress를 활용한 단위/E2E 테스트를 도입하여 서비스의 안정성을 확보할 예정입니다.
4. **SEO 및 성능 최적화**: Next.js로의 전환 또는 SSR 적용을 통해 검색 엔진 최적화와 초기 로딩 속도를 개선할 계획입니다.
