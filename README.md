# ☕ Nada Coffee Frontend Project

나다커피(Nada Coffee)의 신선한 원두와 음료를 만나볼 수 있는 프리미엄 커머스 플랫폼입니다.

---

## 🚨 백엔드 개발자 요청 사항 (Urgent Requests)

원활한 기능 완성을 위해 다음 API 및 DB 설정 변경을 요청드립니다.

### 1. 장바구니 및 주문 (Cart & Order)
- **상품 옵션 필수 생성**: 현재 `optionId`가 외래키 제약 조건으로 묶여 있어, 옵션이 없는 상품은 장바구니 담기 및 주문이 불가능합니다. 모든 상품에 **최소 한 개의 기본 옵션**을 생성해 주시거나, `optionId`를 **Optional(null 허용)**로 변경 부탁드립니다.
- **주문 생성 API 검증 완화**: `POST /api/orders` 시 `optionId`가 누락되어도 주문이 생성될 수 있도록 처리가 필요합니다.

### 2. 고객의 소리 (Contact Us)
- **Payload 용량 상향**: Tiptap 에디터를 통해 **이미지를 Base64 형태**로 전송하고 있습니다. 서버의 **JSON Body Size Limit을 5MB~10MB**로 상향 요청드립니다. (현재 이미지 포함 시 500 에러 발생 가능성)
- **HTML 태그 허용**: 본문의 HTML 태그(`p`, `img`, `strong` 등)가 필터링되지 않고 DB에 그대로 저장되도록 설정 확인 부탁드립니다.

### 3. 이벤트 상세 (Event Detail)
- **상세 조회 API**: 특정 이벤트의 상세 내용(긴 이미지 URL 등)을 가져올 수 있는 `GET /api/events/{id}` API 구현을 요청드립니다.

## 📅 작업 현황 (2026-02-03)

### ✅ 오늘 완료한 작업 (Today's Achievements)

#### 1. 상품 상세 페이지 및 구매 로직 개편 ☕
- **프리미엄 상세 페이지 구축**: 이미지 갤러리(썸네일 전환), 영양 성분 가이드, 제품 특징 섹션 구현.
- **Hot/Ice 옵션 시스템**: 온도 선택에 따른 **실시간 금액 가산(+500원)** 및 총 결제 금액 자동 계산 로직 도입.
- **구매 프로세스 최적화**: '장바구니 담기'와 '바로 구매하기' 버튼을 분리하여 직관적인 UX 제공.

#### 2. 마이페이지(My Page) 기능 완성 👤
- **내 정보 관리**: 실시간 정보 조회, 회원 정보 수정(이름/연락처), 비밀번호 변경 API 연동 완료.
- **주문 내역 시스템**: 전체 주문 목록 조회 및 **주문 상세 모달** 구현. 주문 취소 요청 API 연동.
- **포인트 시스템**: 보유 포인트 잔액 조회 및 **상세 이용 내역(적립/사용)** 리스트 구현.

#### 3. 결제 시스템 최종 통합 (Toss Payments) 💳
- **주문서 선 생성 로직**: 결제창 호출 전 서버에 주문 데이터를 먼저 저장하고, 발급된 `orderId`를 토스 결제와 매칭.
- **최종 승인 자동화**: 결제 성공 후 `SuccessPage`에서 서버 승인 API(`confirm`)를 자동 호출하여 결제 프로세스 완결.

#### 4. 고객 지원 서비스 고도화 (Support) 💬
- **Tiptap 웹 에디터 도입**: 문의하기 본문에 풍부한 텍스트 편집 및 **이미지 드래그 앤 드롭 업로드** 기능 추가.
- **문의 관리 시스템**: 작성 완료 후 성공 모달 노출 및 **내 문의 내역 리스트/상세/수정/삭제(CRUD)** 기능 구현.
- **사용자 편의성**: 로그인 정보 기반 작성자 이름/이메일 자동 세팅.

#### 5. 메인 페이지 및 뉴스 개편 📰
- **가맹점 찾기 섹션**: `assets/shop` 이미지를 활용한 카드형 리스트 및 페이드인/호버 스케일 애니메이션 적용.
- **뉴스 게시판**: 썸네일 이미지가 포함된 프리미엄 카드 리스트로 개편. 제목/내용 필터링 검색 기능 추가.

## 🛠 Tech Stack
- **Frontend**: React 19, TypeScript, Vite
- **Editor**: Tiptap (Rich Text Editor)
- **Styling**: Tailwind CSS v4, Framer Motion
- **State Management**: Zustand, TanStack Query v5
- **Integration**: Toss Payments, Daum Postcode API
