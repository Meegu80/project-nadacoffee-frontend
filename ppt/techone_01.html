기술 스택
Part 1. Core & View

프론트엔드 핵심 라이브러리와 전역 상태 관리, 그리고 UI의 뼈대를 잡는 스타일링 기술입니다. 무한 스크롤 패턴을 도입하여 데이터 렌더링 성능을 최적화했습니다.

1. 코어 및 프레임워크

  • React (v19): 사용자 인터페이스 구축 핵심 라이브러리
  • TypeScript: 정적 타입 체크를 통한 코드 안정성 확보
  • Vite: 빠르고 가벼운 프론트엔드 빌드 툴
  • React Router (v7): 클라이언트 사이드 라우팅

2. 상태 관리

  • Zustand: 가볍고 유연한 전역 상태 관리 라이브러리
  • TanStack React Query: 서버 상태 관리, 캐싱 및 동기화

3. 스타일링 및 애니메이션

  • Tailwind CSS (v4): 유틸리티 퍼스트 CSS 프레임워크
  • tailwind-merge: Tailwind CSS 클래스 병합 유틸리티
  • Framer Motion: 부드럽고 복잡한 UI 애니메이션 구현
