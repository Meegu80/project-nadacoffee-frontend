Inventory Intelligence #03

Low Stock Signal System

MenuPage.tsx (STOCK) 
// [판단 기준] 옵션별 재고 합산 및 품절 임박 구간(1~5개) 산출
const totalStock = product.options?.reduce((sum, opt) => 
  sum + opt.stockQty, 0) ?? 0;

const isLowStock = totalStock > 0 && totalStock <= 5;

// [UI 구현] Interactive CSS Keyframes
{isLowStock && (
  <div className="absolute top-4 right-4 bg-orange-500 
    animate-bounce shadow-lg px-2 py-1 rounded-lg flex 
    items-center gap-1">
    <AlertCircle size={12} />
    <span className="text-[10px] font-black">품절임박</span>
  </div>
)}
다중 옵션 재고 추적
단품 데이터가 아닌, 여러 세부 옵션(Hot/Ice 등)의 재고 총량을 `reduce` 함수로 정밀하게 합산하여 정확한 정보를 도출.
시각적 긴박감 유도
Tailwind의 `animate-bounce` 애니메이션을 배지에 직접 적용하여 사용자의 시선을 자연스럽게 유도하고 구매 전환율을 향상.
재고가 0일 때는 SOLD OUT 레이어로 교체되며, 1~5개 사이일 때만 이 다이내믹 배지가 활성화됨.
