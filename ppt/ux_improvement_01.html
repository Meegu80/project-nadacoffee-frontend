UX/UI Improvement Case #01

시각적 의사결정 가이드

MenuPage.tsx (Logic) 
// [핵심 UX 로직] 정보 우수성 및 위계 설정
const totalStock = calculations.sumStock(product);
const isSoldOut = totalStock === 0;
const isLowStock = totalStock > 0 && totalStock <= 5;

// 중첩되는 배지 사이에서 최적의 정보 노출 우선순위 설정
return (
  <ProductCard>
    {isSoldOut ? (
      <SoldOutOverlay /> // 1순위: 구매 불가 상태 명시
    ) : (
      <>
        {isLowStock && <LowStockBadge />} // 2순위: 재고 부족 알림 (희소성 강조)
        {isHot && <HotBadge />} // 3순위: 인기 상품 표시 (신뢰 데이터)
        {isNew && <NewBadge />} // 4순위: 신규 출시 안내
      </>
    )}
  </ProductCard>
);
인지 부하 감소 (Cognitive Load Reduction)
데이터 기반 인지 유도
사용자가 모든 상품을 일일이 클릭해 재고를 확인하지 않아도, 리스트 수준에서 구매 가능 여부와 인기 정도를 즉각 파악하도록 설계했습니다.
정보 위계 (Hierarchy)
우선순위 렌더링 시스템
배지가 중첩되어 시각적으로 혼란을 주지 않도록 로직 상에서 상호 배타적인 렌더링 우선순위(Sold Out > Low Stock)를 적용했습니다.
행동 유도 (Nudging)
실시간 재고 연동 애니메이션
품절 임박 상품에 바운스 애니메이션을 적용하여 사용자의 시선을 자연스럽게 유도, 빠른 의사결정을 돕는 넛지 UX를 구현했습니다.
