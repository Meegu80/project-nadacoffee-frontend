UX/UI Improvement Case #02

방해요소 최소화한 검색 경험

useDebounce.ts & Search.tsx 
// [핵심 UX 로직] 불필요한 서버 통신 최소화 및 반응성 확보
function useDebounce(value, delay) {
  const [debouncedValue, setDebouncedValue] = useState(value);

  useEffect(() => {
    const handler = setTimeout(() => {
      setDebouncedValue(value);
    }, delay);

    return () => clearTimeout(handler);

  }, [value, delay]);

  return debouncedValue;
}

// 메뉴 검색창에서의 활용

const debouncedQuery = useDebounce(searchQuery, 300);

useEffect(() => {
  if (debouncedQuery)
    fetchResults(debouncedQuery);
}, [debouncedQuery]);
시스템 반응성 (System Responsiveness)
지능형 입력 지연 처리 (Debouncing)
매 입력마다 서버에 요청을 보내지 않고, 사용자의 타이핑이 멈춘 시점을 감지하여 데이터를 패칭함으로써 서버 부하는 줄이고 체감 속도는 높였습니다.
연속적 경험 (Continuous Flow)
실시간 자동완성 피드백
검색 버튼을 누르는 행위를 제거하고, 입력에 반응하는 Glassmorphism 자동완성 레이어를 제공하여 흐름이 끊기지 않는 검색 경험을 구축했습니다.
Performance UX
검색 결과가 없을 때의 빈 화면 대신 '검색 중...' 스켈레톤 UI를 제공하여 로딩 중에도 앱이 작동하고 있음을 사용자가 인지하게 합니다.
