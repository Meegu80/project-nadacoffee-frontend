Perceived Speed Improvement
데이터가 로드되기 전 실제 레이아웃과 유사한 형태를 미리 보여줌으로써, 로딩 스피너보다 시스템이 빠르게 동작하고 있다는 심리적 안도감 제공.
Layout Stability (CLS)
상품 정보가 로드된 후에도 레이아웃이 급격하게 변하지 않도록(Shift) 고정된 비율의 이미지 영역과 텍스트 영역을 선점하여 시각적 안정성 확보.
Lightweight CSS Animation
Tailwind CSS의 `animate-pulse`를 활용하여 별도의 무거운 JS 로직 없이도 세련된 쉬머(Shimmer) 효과를 구현, CPU 리소스 최적화.
나다커피의 카드 비율(3:4)과 라운드 값(20px, 24px)을 스켈레톤에 동일하게 적용하여 데이터 로드 전후의 이질감을 최소화함.
