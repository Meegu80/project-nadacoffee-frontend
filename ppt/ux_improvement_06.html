콘텐츠 몰입을 돕는 실시간 진행 표시

ScrollProgress.tsx 
// Framer Motion을 활용한 60fps 스크롤 인터랙션
const ScrollProgress: React.FC = () => {
  const { scrollYProgress } = useScroll();
  
  // 탄성 있는 스프링 애니메이션으로 심리적 만족감 제공
  const scaleX = useSpring(scrollYProgress, {
    stiffness: 100, damping: 30, restDelta: 0.001
  });

  return (
    <motion.div
      className="fixed top-0 left-0 right-0 h-2"
      style={{ scaleX, background: '#FFD400', zIndex: 100 }}
    />
  );
};
가시적 가이드 (Visual Guidance)
보이지 않는 끝을 시각화
긴 상세 페이지나 브랜드 스토리에서 사용자가 느낄 수 있는 막연함을 해소하고, 현재 위치를 명확히 인지시켜 이탈률을 낮췄습니다.
심리적 보상 (Psychological Reward)
부드러운 스프링 인터랙션
단순한 선의 확장이 아닌, 탄성이 느껴지는 스프링 물리 엔진을 적용하여 스크롤 행위 자체에 '재미'와 '만족감'을 부여했습니다.
Identity Overlay
최상단에 위치한 브랜드 옐로우 라인은 웹사이트 전체에 일관된 정체성을 부여하며 프리미엄 서비스임을 끊임없이 상기시킵니다.
