MenuPage.tsx (Interaction Logic) 
// 1. Intersection Observer를 활용한 가시성 감지
useEffect(() => {
  if (!isAllCategory) return; // '전체' 카테고리에서만 활성화

  const observer = new IntersectionObserver((entries) => {
    if (entries[0].isIntersecting) {
      // 하단 도달 시 표시 아이템 개수(visibleCount) 증가
      setVisibleCount((prev) => 
        Math.min(prev + itemsPerPage, filteredProducts.length)
      );
    }
  }, { threshold: 0.1 });

  if (observerTarget.current) {
    observer.observe(observerTarget.current);
  }
  return () => observer.disconnect();
}, [isAllCategory, filteredProducts.length]);

// 2. 하단 로딩 트리거 (Target Element)
<div ref={observerTarget} className="w-full h-20">
  {visibleCount < filteredProducts.length && <SkeletonGrid />}
</div>
Intersection Observer API
스크롤 시 하단 `observerTarget`이 화면에 노출되는 시점을 감지하여 다음 페이지의 데이터를 로드하거나 렌더링 범위를 확장함.
Hybrid UX Strategy
전체 메뉴 로딩 시 'Batch Prefetching'과 'Infinite Scroll'을 결합하여, 초기 렌더링 속도와 연속적인 사용자 스크롤 경험을 동시에 만족.
Contextual Loading
상세 카테고리에서는 페이지네이션을, '전체' 카테고리에서는 무한 스크롤을 제공하는 하이브리드 인터페이스를 통해 효율적인 데이터 탐색 지원.
불필요한 리렌더링을 방지하기 위해 `IntersectionObserver`의 클린업 기능을 철저히 구현하여 성능 최적화 달성.
