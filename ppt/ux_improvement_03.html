UX/UI Improvement Case #03

감성적인 피드백 루프

GlobalAlert & Framer Motion 
// [핵심 UX 로직] 일관된 알림 피드백 및 부드러운 전환
const { showAlert } = useAlertStore();

// 1. 비동기 작업 후 명확한 상태 피드백 제공 (Zustand 연동)
try {
  await mutation.mutateAsync(data);
  showAlert("문의가 정상 접수되었습니다.", "성공", "success");
} catch(e) {
  showAlert("일시적인 오류가 발생했습니다.", "오류", "error");
}

// 2. Framer Motion을 활용한 심리적 안정감 부여
<AnimatePresence mode="wait">
  <motion.div 
    initial={{ opacity: 0, y: 20 }}
    animate={{ opacity: 1, y: 0 }}
    exit={{ opacity: 0, scale: 0.95 }}
  >
    {/* 페이지 콘텐츠 */}
  </motion.div>
</AnimatePresence>
일관성 (Consistency)
중앙 집중형 알림 시스템
모든 페이지에서 동일한 톤앤매너의 애니메이션 알림(Toast)을 제공하여 시스템에 대한 신뢰성과 예측 가능성을 높였습니다.
심리적 안정감 (Mental Comfort)
부드러운 상태 전환 (Smoothing)
데이터가 바뀔 때 화면이 툭툭 끊기지 않도록 `AnimatePresence`를 적용, 실제 사물이 움직이는 듯한 물리적 질감을 재현했습니다.
Delightful Experience
성공 모달의 체크 애니메이션, 메뉴 호버 시의 미세한 스케일링 등 디테일한 마이크로 인터랙션이 브랜드에 대한 긍정적 경험을 형성합니다.
