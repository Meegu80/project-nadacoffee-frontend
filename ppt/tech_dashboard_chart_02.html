Recharts Data Visualization

Responsive Area Chart

AdminDashboard.tsx (Recharts UI) 
// 1. 그라디언트 및 리스폰시브 컨테이너 설정
<ResponsiveContainer width="100%" height="100%">
    <AreaChart data={chartData}>
    <defs>
      <linearGradient id="colorSales" x1="0" y1="0" x2="0" y2="1">
        <stop offset="5%" stopColor="#FFD400" stopOpacity={0.2}/>
        <stop offset="95%" stopColor="#FFD400" stopOpacity={0}/>
      </linearGradient>
    </defs>
    
    // 2. 차트 구성 요소 (축, 그리드, 툴팁)
    <XAxis dataKey="name" tick={{ fontSize: 14 }} />
    <YAxis tickFormatter={(val) => `₩${val/1000}k`} />
    <Area type="monotone" dataKey="sales" stroke="#FFD400" strokeWidth={4} fill="url(#colorSales)" />
  </AreaChart>
</ResponsiveContainer>
Responsive Design
`ResponsiveContainer`를 사용하여 브라우저 크기 변화에 따라 차트의 너비와 높이가 자동으로 리사이징되어 완벽한 대시보드 레이아웃 유지.
UI 시각화 기법
영역 차트(`AreaChart`) 하단에 커스텀 선형 그라디언트를 적용하여 단순한 선 그래프보다 풍성하고 깊이감 있는 시각 효과 구현.
데이터 가독성 최적화
Y축의 큰 금액 데이터를 `tickFormatter`로 단위를 요약(k) 표시하고, 툴팁에서 정확한 원화를 포맷팅해 보여줌으로써 명확한 통찰력 제공.
`monotone` 타입의 곡선을 사용하여 급격한 매출 변화를 부드럽게 연결, 추세를 직관적으로 파악할 수 있도록 디자인함.
