사이트 개발 기획서

기술 스택
Part 1. Core & View

프론트엔드 핵심 라이브러리와 전역 상태 관리, 그리고 UI의 뼈대를 잡는 스타일링 기술입니다. 무한 스크롤 패턴을 도입하여 데이터 렌더링 성능을 최적화했습니다.

1. 코어 및 프레임워크

  • React (v19): 사용자 인터페이스 구축 핵심 라이브러리
  • TypeScript: 정적 타입 체크를 통한 코드 안정성 확보
  • Vite: 빠르고 가벼운 프론트엔드 빌드 툴
  • React Router (v7): 클라이언트 사이드 라우팅

2. 상태 관리

  • Zustand: 가볍고 유연한 전역 상태 관리 라이브러리
  • TanStack React Query: 서버 상태 관리, 캐싱 및 동기화

3. 스타일링 및 애니메이션

  • Tailwind CSS (v4): 유틸리티 퍼스트 CSS 프레임워크
  • tailwind-merge: Tailwind CSS 클래스 병합 유틸리티
  • Framer Motion: 부드럽고 복잡한 UI 애니메이션 구현

기술 스택
Part 2. UI & Experience

폼 유효성 검사, 시각적 차트/슬라이더, 확장형 에디터 등 사용자 경험을 극대화하는 요소들입니다. 스켈레톤 UI를 통해 로딩 화면의 사용성을 개선했습니다.

4. 폼 및 유효성 검사

  • React Hook Form: 성능이 뛰어나고 유연한 폼 상태 관리
  • Zod: TypeScript 기반의 스키마 선언 및 유효성 검증
  • @hookform/resolvers: React Hook Form과 Zod 연동

5. UI 컴포넌트 & 아이콘

  • Lucide / React Icons: 다양하고 사용하기 쉬운 아이콘 팩
  • Swiper: 모바일 터치를 지원하는 강력한 슬라이더/캐러셀
  • Recharts: React 기반의 조합 가능한 차트 라이브러리
  • Hot Toast & Helmet Async: 알림(Toast) 및 SEO 관리
  • 스켈레톤 UI (Skeleton UI): Placeholder 노출로 체감 대기시간 감소

6. 에디터 (Rich Text Editor)

  • Tiptap: Headless 기반의 강력하고 확장 가능한 텍스트 에디터
  • 사용 확장 (Extensions):
    - Starter Kit, Image, Link, Placeholder, Text Align, Color 등

기술 스택
Part 3. Integration & Tools

외부 결제 및 지도 API 연동, 데이터 및 보안 처리를 위한 유틸리티, 그리고 안정적인 개발을 보장하는 테스트 및 IDE 환경입니다.

7. 결제 및 지도/주소

  • Toss Payments SDK: 간편결제 및 일반 결제 연동
  • React Kakao Maps SDK: 카카오 지도 API 래퍼
  • React Daum Postcode: 우편번호/주소 검색 서비스 연동

8. 유틸리티

  • Axios: Promise 기반의 HTTP 클라이언트 라이브러리
  • Day.js: 가볍고 날짜 및 시간 형식 지정을 위한 라이브러리
  • JWT Decode: JWT 토큰 페이로드 디코딩 유틸리티
  • DOMPurify: XSS 공격 방지를 위한 HTML 문자열 정화

9. 테스트 및 개발 도구

  • Storybook: UI 컴포넌트 독립 개발 및 문서화 환경
  • Vitest / Jest / E2E: 빠르고 강력한 테스트 환경 구축
  • ESLint & Prettier: 코드 컨벤션 유지 및 포맷팅 자동화
  • WebStorm: 강력한 개발 통합 환경 (IDE)
  • Antigravity: 파이썬 기반의 가상환경 및 패키지 관리 도구
