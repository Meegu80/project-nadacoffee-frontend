Experience

Intersection Observer 무한 스크롤

사용자가 스크롤을 멈추지 않고 상품을 탐색할 수 있도록 비동기 데이터 로딩을 적용했습니다.

const observer = new IntersectionObserver((entries) => {
  if (entries[0].isIntersecting) {
    loadMoreItems(); // 자연스러운 추가 로딩
  }
});
