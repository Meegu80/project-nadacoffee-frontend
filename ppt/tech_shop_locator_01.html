Kakao Map Auto-Bounds Logic

SearchShop.tsx (Bounds Logic) 
// 1. 마커 초기화 및 영역 객체(Bounds) 생성
const bounds = new window.kakao.maps.LatLngBounds();
markersRef.current.forEach(m => m.setMap(null));

// 2. 검색 결과 루프를 통한 마커 생성 및 영역 확장
filteredShops.forEach(shop => {
  const pos = new window.kakao.maps.LatLng(shop.lat, shop.lng);
  const marker = new window.kakao.maps.Marker({ 
    position: pos, map: map 
  });
  markersRef.current.push(marker);
  // 영역(Bounds)에 현재 좌표 포함
  bounds.extend(pos); 
});

// 3. 결과 수에 따른 지도 뷰포인트 최적화
if (filteredShops.length === 1) {
  map.setCenter(pos); map.setLevel(3);
} else if (filteredShops.length > 0) {
  // 모든 마커가 보이도록 지도 자동 범위 조정
  map.setBounds(bounds);
}
지능형 영역 제어 (Auto-Bounds)
단순히 특정 좌표로 이동하는 것이 아니라, `LatLngBounds` 객체를 통해 검색된 모든 매장이 한 화면에 들어올 수 있도록 지도의 중심과 레벨을 자동 계산하여 최적화함.
메모리 누수 방지 (Ref 관리)
검색어가 바뀔 때마다 `markersRef`에 저장된 이전 마커들을 전체 순회하며 `setMap(null)` 처리함으로써 브라우저 메모리 낭비와 마커 중첩 현상을 완벽히 방지함.
결과가 1개일 때는 `setLevel(3)`을 통해 매장 상세 정보를 볼 수 있는 근거리 줌을 적용하고, 여러 개일 때는 광역 뷰를 제공하는 비즈니스 UX 로직 적용.
