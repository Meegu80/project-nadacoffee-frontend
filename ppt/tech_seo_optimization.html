Smart SEO Management

SEO.tsx / Home.tsx 
// 1. 공통 SEO 컴포넌트 설계
const SEO = ({ title, description }) => (
  <Helmet>
    <title>{title} | NADA COFFEE</title>
    <meta name="description" content={description} />
    // Open Graph Protocol - SNS 공유 최적화
    <meta property="og:title" content={title} />
    <meta property="og:type" content="website" />
  </Helmet>
);

// 2. 페이지별 유연한 메타데이터 주입
function Home() {
  return (
    <div>
      <SEO 
        title="홈" 
        description="커피를 커피답게, 나다커피" 
      />
      ...
    </div>
  );
}
Dynamic Metadata (Helmet)
`react-helmet-async`를 활용하여 SPA 환경에서도 각 페이지 접속 시 브라우저 탭의 제목과 description 메타 태그를 동적으로 교체하여 검색 로봇의 수집 효율 극대화.
OG (Open Graph) Support
카카오톡, 페이스북 등 외부 SNS에서 링크 공유 시 사이트 정보가 미리보기 카드 형태로 깔끔하게 노출되도록 OG 태그를 표준 명세에 맞춰 자동 생성 및 관리.
Structural SEO Strategy
재사용 가능한 `SEO` 공통 컴포넌트를 설계하여 모든 페이지에 일관된 메타데이터 주입 체계를 구축하고, 마크업 최적화를 통해 사이트 구조 안정성 확보.
단순 디자인을 넘어, 유통 및 마케팅 관점에서 고객 유입을 극대화할 수 있는 '기술적 마케팅' 기반의 프론트엔드 환경 구축에 집중함.
