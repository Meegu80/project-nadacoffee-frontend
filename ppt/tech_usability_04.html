데이터 무마찰 경험

TanStack Query의 캐싱 전략(`staleTime`, `gcTime`)을 최적화하고 주요 상세 페이지의 데이터를 Pre-fetching하여 즉각적인 응답성을 구현했습니다.

const { data } = useQuery({
  queryKey: ['products', category],
  queryFn: fetchProducts,
  staleTime: 1000 * 60 * 5, // 5분간 캐싱
  gcTime: 1000 * 60 * 10
});
