SearchShop.tsx (Interaction) 
// 1. 리스트 아이템 클릭 시 지도 동기화 로직
{filteredShops.map(shop => (
  <div 
    key={shop.id} 
    onClick={() => {
      // 부드럽게 해당 좌표로 중심 이동 (PanTo)
      map.panTo(new window.kakao.maps.LatLng(shop.lat, shop.lng));
    }}
    className="p-6 cursor-pointer hover:bg-gray-50"
  >
    <h3>{shop.name}</h3>
    <p>{shop.address}</p>
  </div>
))}

// 2. 지도 초기화(Reset) 및 검색 파라미터 제어
const handleReset = () => {
  setSearchQuery("");
  setFilteredShops(MOCK_SHOPS);
  setSearchParams({}); // URL 쿼리 파라미터 초기화
};
뷰 스테이트 동기화 (Sync)
매장 리스트와 지도 영역을 분리하되, 리스트 클릭 시 `map.panTo` API를 사용하여 지도가 목표 좌표로 부드럽게 슬라이드 이동하는 고급 UX 제공.
URL 파라미터 연동
`useSearchParams`를 사용하여 공유된 링크(예: ?search=강남)만으로도 즉시 검색 결과가 지도에 렌더링되도록 딥링킹(Deep Linking) 구조 지원.
좌측 리스트와 우측 지도의 7:3 비율 레이아웃 및 커스텀 스크롤바를 통해 데스크탑과 모바일 모두에서 쾌적한 사용성을 확보함.
