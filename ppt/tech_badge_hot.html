Visual Identity #01

HOT Best Seller Badge

MenuPage.tsx (HOT) 
// [판단 기준] 실시간 베스트셀러 TOP 10 포함 여부
const { top10Ids } = useBestSellers();
const isHot = !isSoldOut && top10Ids.includes(product.id);

// [UI 구현] SVG Curve Text Stamp
{isHot && (
  <div className="absolute bottom-6 left-6">
    <svg width="110" height="110" viewBox="0 0 100 100">
      // 텍스트가 흐를 곡선 경로 정의
      <path id="stampCurve" d="M 20,50 A 30,30 0 1,1 80,50" />
      <textPath href="#stampCurve">NADA BEST SELLER</textPath>
      <text x="50" y="62" fontSize="28">HOT</text>
    </svg>
  </div>
)}
데이터 기반 판별
단순 수동 지정이 아닌, 주문 통계 로직인 `useBestSellers` 커스텀 훅을 통해 실시간 TOP 10 상품을 자동으로 판별하여 배지를 노출.
SVG Typography
이미지 파일을 사용하지 않고 SVG `textPath`를 활용해 곡선 텍스트를 렌더링함으로써 해상도 저하 없는 선명한 프리미엄 스탬프 효과 구현.
`drop-shadow-2xl` 필터를 결합하여 실제 도장을 찍은 듯한 입체감을 상품 이미지 위에 부여함.
